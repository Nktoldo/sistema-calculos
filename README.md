## Sistema de Calculos

Sistema criado para duas empresas de equipamentos laboratoriais com o objetivo de organizar as vendas e proteger dados da empresa.

Antes, o trabalho era muito manual e "espalhado", o que causava erros frequentes nos cálculos e deixava informações financeiras sigilosas visíveis para todos os funcionários. Além disso, não havia um histórico salvo das propostas antigas.

O sistema resolveu isso centralizando tudo em um só lugar. Agora, os cálculos são feitos automaticamente (sem erros) e existe um controle de permissões: Administradores têm acesso total, enquanto Colaboradores veem apenas o necessário para trabalhar. Tudo fica salvo e organizado com segurança no Firebase.
